# .github
Uniloader — безопасный Telegram-бот, созданный для получения видео с ютуба. Отправьте ссылку, и сервис быстро обработает её, вернув нужный ролик прямо в чат. Работает на всех устройствах, не требует регистрации и сторонних сайтов. Простое взаимодействие, стабильная работа и аккуратный интерфейс.
